# adweave-plugins-stable

Pinned-release marketplace for [AdWeave](https://adweave.ai) Claude Code plugins. This is the **stable channel** — plugins here are pinned to tagged releases and only advance when a new version is cut.

For the bleeding-edge channel that tracks `main`, see [adweave-ai/adweave-claude-plugin](https://github.com/adweave-ai/adweave-claude-plugin) directly.

## Install

In Claude Code:

```
/plugin marketplace add adweave-ai/adweave-plugins-stable
/plugin install adweave@adweave-stable
```

In Cowork / Claude Desktop: browse plugin marketplaces and add `adweave-ai/adweave-plugins-stable`.

## What's inside

| Plugin | Pinned to |
|---|---|
| `adweave` | `adweave-ai/adweave-claude-plugin@v1.1.0` |

## Upgrading

When a new AdWeave plugin release ships, this marketplace's `marketplace.json` is updated to the new tag. Run `/plugin update` in Claude Code (or the equivalent in your host) to pull the new pinned version.
